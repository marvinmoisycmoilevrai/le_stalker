Choix des technologies

Nous prévoyons d'utiliser les technologie suivant (liste non-exhaustive) :
Client :
  -Vue.js             : Spécialisé one page, plus abordable qu'angular (et mieux documenté)
  -BootstrapVue       : Mise en page propre et déjà responsive
  -Google Maps API    : Afficher les données sur la carte
Serveur :
  -Node.js            : Fonctionne mieux que php avec les API REST, c'est également ce que l'on découvre cette année
  -fb NodeJS wrapper  : Récupérer les données Facebook (API REST)
  -Express            : Faire transiter les informations entre MySQL, node.js et vue.js
  -MySQL              : Stocker les données
  -IP geolocation API : Pouvoir traquer la géolocalisation de quelqu'un ayant désactivé la sienne
